Principais Formatações

Negrito / Destaque

Nessa frase temos um termo em Negrito. Porém essa tag <b> é uma tag não semântica.

Nesta frase temos um termo em Negrito usando <strong>. Que é uma tag semântica, além de visualmente ser a mesma aparência, ela dá o significado de forte.

Itálico / Ênfase

Nessa frase temos um texto itálico usando a tag <i>, que não é semântico.

Nessa frase temos um texto em ênfase usando a tag <em>, que é semântico

Texto Marcado

Podemos criar também um texto marcado, como este aqui, com a tag <mark>.

E em outro parágrafo temos outro texto marcado no final

Texto Grande e Pequeno

Estamos Criando um texto grande e um texto pequeno. A tag <big> está obsoleta, porém <small> não está.

Texto Deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado. Isso com a tag <del>.

Texto Inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois. Isso com a tag <ins>

Existe também o sublinhado com a tag <U>, que é não semântica.

Texto sobrescrito

Para inserir coisas do tipo x20+3. Com a tag <sup>.

Texto subscrito

Para inserir coisas do tipo H2O. Com a tag <sub>.
