Parágrafos e quebras de linha

Você pode escrever um parágrafo de qualquer jeito.
Basta colocar tudo dentro do par de tags <p> e </p>

Se precisar quebrar um texto em algum lugar específico,
como este, você pode usar a tag <br>.

Vamos adicionar alguns símbolos especiais:
® ©
™
€ £ ¥ ¢
Δ
↑

Vamos adicionar alguns emojis:
🐕 🐈

Para adicionar emoji vc precisa colocar &# e então o código do emoji.
Aqui está o site recomendado pelo Guanabara para pegar os códigos de emojis: EmojiPedia
