CSS Inline

IMPORTANTE

CSS inline não é recomendado, ele utiliza CSS dentro de tags no HTML, como <body>, <h1>, etc. Usa o CSS bem pontual, qualquer mudança terá que ser feita em todas as outras tags uma por uma.
